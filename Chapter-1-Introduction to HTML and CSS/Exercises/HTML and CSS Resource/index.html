HTML and CSS Resource Guide

HTML: Hypertext Markup Language

Heading <h1> to <h6>

Adds a heading and with a different sizes

Input:

                
                <h1>Heading 1</h1>
                <h3>Heading 2</h3>
                <h4>Heading 3</h4>
                <h4>Heading 4</h4>
                <h5>Heading 5</h5>
                <h6>Heading 6</h6>

Output:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Paragraph <p>

Adds a paragraph

                
                    <p>This is a paragraph</p>

Comment <--...-->

Serves as a explanation or information about what does the line of code do for other programmers

Input:

                
                    <h1>Heading 1</h1> <!--This is a heading tag--!>

Anchor <a href="https://">

Adds a hyperlink, which can link and open another page or a different website

It is also accompanied by a href attribute which is the url of the page

Input:

                
                    <a href="https://bathspa.ac.ae/">Bath Spa University RAK Website</a>

Horizontal Line <hr>

Adds a horizontal line in between html elements

Input:

                
                    <h1>Heading 1</h1>
                    <hr>
                    <p>Above me is a horizontal line</p>

Line Break <br>

Adds a line break in betweeen text which moves it to a new line

Input:

                
                    <p>The text will move<br>to another line</p>

Subscript <sub>

Makes a text subscript

Input:

                
                    <p>This is the chemical formula for water: H<sub>2<sub>O</p>

Superscript <sup>

Makes a text superscript

Input:

                
                    <p>10<sup>2<sup>= 100</p>

Image <img src="" alt="">

Displays an image which is accompanied by the src attribute which is the url or filenamme of the image and an alt attribute which is used for screen readers or when the image doesn't load properly

Input:

                
                    <img src="images/dog.jpg" alt="Dog">

Lists <ul> & <ol>

Unordered lists are displayed with bullet points

Ordered lists are displayed with numbers

Input:

                
                    <ul>
                        <li>Apples</li>
                        <li>Oranges</li>
                        <li>Strawberries</li>
                    </ul>

                    <ol>
                        <li>First</li>
                        <li>Second</li>
                        <li>Third</li>
                    </ol>

CSS: Cascading Style Sheets

Color color: desired color;

Changes the text color

Input:

HTML
                    
                        <p>This text is color green</p>
CSS
                    
                        p {
                            color: green;
                        }

Font Size font-size: desired text size;

Changes the font size of a text

Input:

HTML
                    
                        <p>This text is a little too big</p>
CSS
                    
                        p {
                            font-size: 40px;
                        }

Text Transform text-transform: desired capitalization;

Changes the capitalzation of a text

Input:

HTML
                    
                        <p>MAKE THIS ALL LOWERCASE LETTERS</p>
CSS
                    
                        p {
                            text-transform: lowercase;
                        }
